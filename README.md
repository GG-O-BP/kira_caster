# kira_caster

씨미(ci.me) 방송에서 쓸 수 있는 챗봇이야~!

채팅창에 명령어를 치면 키라캐스터가 대답해주는 거야! 출석체크도 하고, 포인트도 모으고, 미니게임도 할 수 있어!! 진짜 재밌겠지?!

## 이런 걸 할 수 있어!

| 기능 | 설명 |
|------|------|
| 출석체크 | `!출석` 치면 오늘 출석 완료~! |
| 포인트 | 방송 보면서 포인트 모을 수 있어! |
| 미니게임 | `!게임` 으로 간단한 게임을 할 수 있어! |
| 채팅 필터 | 나쁜 말은 자동으로 걸러줘! 착한 채팅만~ |

## 시작하는 방법

Gleam이랑 Erlang/OTP가 설치되어 있어야 해!

```sh
gleam deps download   # 필요한 것들 다운받기
gleam build           # 빌드하기
gleam run             # 실행하기!
```

실행하면 이렇게 나와!

```
[mock] Connected
kira_caster started with mock adapter
[mock] Sending: Bot is online!
```

지금은 연습용(mock) 모드로 돌아가는 거야~ 씨미 API가 공개되면 진짜 방송이랑 연결할 수 있어!
