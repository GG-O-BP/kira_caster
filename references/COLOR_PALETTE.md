# kira_caster 컬러 팔레트

kira_caster 대시보드 및 UI에 사용하는 컬러 팔레트.

## 메인 컬러

| 이름 | HEX | 용도 |
|------|-----|------|
| 스트로베리 핑크 | `#FD719B` | 메인 포인트, 버튼, 강조 |
| 코랄 오렌지 | `#FD9371` | 보조 포인트, 그라디언트 끝 |
| 라이트 핑크 | `#FD99B8` | 보조 핑크 |

## 그라디언트

| 이름 | 값 | 용도 |
|------|-----|------|
| 메인 그라디언트 | `linear-gradient(92.54deg, #FF608F 5.84%, #FBB35E 95.21%)` | 주요 버튼, 배너 |
| 보조 그라디언트 | `linear-gradient(92.54deg, #FD719B 5.84%, #FD9371 95.21%)` | 보조 강조 |

## 기능 컬러

| 이름 | HEX | 용도 |
|------|-----|------|
| 그린 | `#00C199` | 성공, 활성 상태 |
| 옐로우 | `#F8C03A` | 경고, 알림 |
| 레드 | `#F77061` | 에러, 위험 |
| 라이트 블루 | `#3B9FFA` | 링크, 정보 |
| 블루 | `#007AFF` | 네비게이션, 페이지네이션 |

## 중립 컬러

| 이름 | HEX / RGBA | 용도 |
|------|------------|------|
| 블루 그레이 | `#54577A` | 본문 텍스트 |
| 라이트 그레이 | `#E9EAEE` | 비활성 요소, 구분선 |
| 화이트 | `#FFFFFF` | 배경, 카드 |
| 블랙 | `#000000` | 강조 텍스트 |
| 오버레이 (연) | `rgba(0,0,0,0.2)` | 연한 그림자 |
| 오버레이 (중) | `rgba(0,0,0,0.5)` | 모달 배경 |
| 오버레이 (진) | `rgba(0,0,0,0.8)` | 진한 오버레이 |
| 글래스 (연) | `rgba(255,255,255,0.1)` | 글래스모피즘 |
| 글래스 (중) | `rgba(255,255,255,0.75)` | 반투명 카드 |

## 폰트

| 이름 | font-family | 용도 |
|------|-------------|------|
| 본문 | `Hiragino Sans, ui-sans-serif, system-ui, -apple-system, BlinkMacSystemFont, Segoe UI, Roboto, Helvetica Neue, Arial, Noto Sans, sans-serif` | 기본 본문 텍스트 |
| 제목 | `var(--yuruka-font)` | 제목, 헤딩 (커스텀 폰트) |
| 영문 | `var(--quicksand-font)` / `Quicksand` | 영문, 숫자 |
| 숫자 | `Roboto` | DIN 스타일 숫자 |
| 코드 | `ui-monospace, SFMono-Regular, Menlo, Monaco, Consolas, Liberation Mono` | 모노스페이스 |
